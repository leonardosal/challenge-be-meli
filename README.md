## Requisitos para instalação local

* NodeJS versão 8.x ou superior

## Instalação

clone este repósitorio depois:

```
$ cd challenge-be-meli
$ npm install
```

## Rodando a aplicação

```
$ npm start
```

## Rodar testes

```
$ npm test
```
